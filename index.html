GDP vs Happiness Analysis

This project analyzes the relationship between GDP and happiness levels across various countries. The analysis explores how economic factors correlate with subjective well-being and aims to provide insights into what truly contributes to happiness.

Tools Used: Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

Key Insights: The analysis revealed that while wealthier countries tend to report higher happiness scores, there is a diminishing return beyond a certain GDP threshold.
