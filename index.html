My Projects

Here are a few projects that showcase my skills in data analysis:

GDP vs Happiness Analysis

This project analyzes the relationship between GDP and happiness levels across various countries. The analysis explores how economic factors correlate with subjective well-being and aims to provide insights into what truly contributes to happiness.

Tools Used: Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

Key Insights: The analysis revealed that while wealthier countries tend to report higher happiness scores, there is a diminishing return beyond a certain GDP threshold.

Flight Delay Prediction

This project predicts flight delays based on historical data. By analyzing factors like weather, airline, and flight schedule, I was able to build a predictive model.

Tools Used: Python, Scikit-learn, Pandas, Matplotlib

Key Insights: The model successfully predicted delays with an accuracy rate of 85%, especially with respect to severe weather conditions.

Retail Sales Forecasting

This project involved forecasting retail sales for a store based on historical sales data and external factors like holidays, promotions, and economic conditions.

Tools Used: Python, Statsmodels, Pandas, Excel

Key Insights: The model accurately predicted sales spikes during promotions and holidays, which helped optimize inventory management.
